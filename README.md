## 📋 نظرة عامة | Overview

**Clean Temp SSD Tool** هي أداة تنظيف متقدمة لنظام Windows، تم تطويرها بواسطة **mhmad240** لمساعدة المستخدمين على تحسين أداء أجهزتهم وتوفير مساحة تخزينية قيمة. تقوم الأداة بتنظيف الملفات المؤقتة والملفات غير الضرورية بأمان تام، مع الحفاظ على استقرار النظام وسلامته.

---

| 8 | إفراغ سلة المحذوفات | Recycle Bin |
| 9 | DISM Component Cleanup | تنظيف مكونات النظام |
| 10 | DISM with Reset Base | تنظيف عميق للنظام |

## 📥 طريقة التحميل والتثبيت | Installation

### الخيار 1: تشغيل ملف Batch مباشرة
1. تحميل ملف `Clean_Temp_SSD_v1.0.bat`
2. الضغط بزر الماوس الأيمن على الملف
3. اختيار **"Run as Administrator"**

### الخيار 2: تشغيل ملف EXE (مفضل)
1. تحميل ملف `Clean_Temp_SSD_v1.0.exe`
2. الضغط بزر الماوس الأيمن على الملف
3. اختيار **"Run as Administrator"**

> ⚠️ **ملاحظة مهمة**: يجب تشغيل الأداة بصلاحيات المدير (Administrator) لتعمل بشكل صحيح.

## 🚀 طريقة الاستخدام | How to Use

1. **شاشة الترحيب**: تعرض معلومات الأداة وإصدارها
2. **التحقق من الصلاحيات**: يتأكد من تشغيلها كـ Administrator
3. **حساب المساحة**: يحسب المساحة المتاحة قبل التنظيف
4. **مراحل التنظيف**: 4 مراحل رئيسية (18 خطوة)
5. **التقارير النهائية**: يعرض المساحة المحررة والنتائج

- ✅ الأداة آمنة تماماً ولا تحذف أي ملفات ضرورية للنظام
- ✅ يمكن استعادة جميع الإعدادات يدوياً إذا لزم الأمر
